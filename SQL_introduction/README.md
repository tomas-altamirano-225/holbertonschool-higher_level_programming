# SQL Introduction

## Description
This project is an introduction to Relational Databases and SQL (Structured Query Language) using MySQL. It covers fundamental concepts of database creation, table management, and basic CRUD operations, establishing the foundation for database administration.

## Topics Covered
* What is a database and what is a relational database.
* What is SQL and MySQL.
* Data Definition Language (DDL) vs Data Manipulation Language (DML).
* Creating databases, tables, and performing SELECT, INSERT, UPDATE, and DELETE operations.
* Using basic SQL functions like COUNT, AVG, MAX, MIN, and GROUP BY to aggregate data.

## Repository Contents
Contains various `.sql` scripts that perform fundamental SQL tasks, starting from listing databases and creating tables to performing aggregations, filtering data, and ordering results.
