# SQL More Queries

## Description
This project dives deeper into SQL using MySQL. It explores more advanced concepts such as user permissions, primary and foreign keys, constraints, subqueries, and various types of JOINs to retrieve complex data structures.

## Topics Covered
* Managing users and granting/revoking privileges in MySQL.
* Establishing relationships between tables using Primary Keys (PK) and Foreign Keys (FK).
* Understanding and applying constraints (`UNIQUE`, `NOT NULL`, `DEFAULT`).
* Using subqueries to solve complex filtering problems.
* Working with `JOIN` operations (INNER JOIN, LEFT JOIN, etc.) to combine and extract data from multiple tables effectively.

## Repository Contents
Contains `.sql` scripts demonstrating user management, schema definitions with relational integrity, and advanced queries involving multiple related tables, aggregations, and conditional logic.
